# MotoCare AstraPay

Vehicle care and maintenance app built with React Native (Expo) for the **AstraPay Hackathon**.

## ⚠️ Status: Prototype (Hackathon Submission)

This is the current **prototyping version** developed for the AstraPay Hackathon. The app demonstrates core vehicle care features including:

- 🏍️ **Multi-Vehicle Support**: Switch between motorbike and car views
- 🤖 **AI Chatbot**: Diagnostic recommendations powered by AI (V-Belt monitoring, battery health alerts)
- 📅 **Service Booking**: Schedule maintenance at AHASS/Auto2000 workshops
- 💳 **AstraPay Integration**: In-app payment flow with PIN authentication
- 📍 **Workshop Locator**: Find nearest Astra Motor service centers

## Tech Stack

- **React Native** (Expo SDK ~56)
- **Expo Vector Icons** (@expo/vector-icons)
- **Animated API** for smooth UI transitions
